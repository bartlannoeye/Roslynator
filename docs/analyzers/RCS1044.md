# RCS1044: Remove original exception from throw statement

| Property                    | Value           |
| --------------------------- | --------------- |
| Id                          | RCS1044         |
| Category                    | Maintainability |
| Default Severity            | Warning         |
| Enabled by Default          | &#x2713;        |
| Supports Fade\-Out          | &#x2713;        |
| Supports Fade\-Out Analyzer | \-              |

## Example

### Code with Diagnostic

```csharp
try
{
    Foo();
}
catch (Exception ex)
{
    throw ex; // RCS1044
}
```

### Code with Fix

```csharp
try
{
    Foo();
}
catch (Exception ex)
{
    throw;
}
```
