# RCS1099: Default label should be the last label in a switch section

| Property                    | Value       |
| --------------------------- | ----------- |
| Id                          | RCS1099     |
| Category                    | Readability |
| Default Severity            | Info        |
| Enabled by Default          | &#x2713;    |
| Supports Fade\-Out          | \-          |
| Supports Fade\-Out Analyzer | \-          |

## Example

### Code with Diagnostic

```csharp
switch (s)
{
    default: // RCS1099
    case "a":
    case "b":
        break;
}
```

### Code with Fix

```csharp
switch (s)
{
    case "b":
    case "a":
    default:
        break;
}
```
